# 🗳️ Voting Eligibility Checker

## 📌 Description
This project is a simple Python-based program 
that checks whether a person is eligible to vote 
based on their age input. It uses conditional 
statements and a while loop to allow multiple 
checks until the user decides to exit.

## 🛠️ Technologies Used
- Python 3.x

## 📦 Installation
No external libraries required.
Just make sure Python is installed on your system.

## ▶️ How to Run
python voting.py

## 📊 Logic Used
- If age >= 18 → Eligible to Vote ✅
- While Loop → Multiple checks allowed
- Break Statement → Exit when user says no

## 💡 Features
- User-friendly input system
- Supports multiple age checks in one run
- Clean and simple code structure
- Beginner friendly Python project
